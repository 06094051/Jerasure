00287 /* Matrix Inversion ------------------------------------------- */
00288 /*
00289    The two matrix inversion functions work on rows*rows matrices of
00290    ints.  If a bitmatrix, then each int will just be zero or one.
00291    Otherwise, they will be elements of gf(2^w).  Obviously, you can
00292    do bit matrices with crs_invert_matrix() and set w = 1, but
00293    crs_invert_bitmatrix will be more efficient.
00294 
00295    The two invertible functions return whether a matrix is invertible.
00296    They are more efficient than the inverstion functions.
00297 
00298    Mat will be destroyed when the matrix inversion or invertible
00299    testing is done.  Sorry.
00300 
00301    Inv must be allocated by the caller.
00302 
00303    The two invert_matrix functions return 0 on success, and -1 if the
00304    matrix is uninvertible.
00305 
00306    The two invertible function simply return whether the matrix is
00307    invertible.  (0 or 1). Mat will be destroyed.
00308  */
00309 
00310 int jerasure_invert_matrix(int *mat, int *inv, int rows, int w);
