00336 /* Matrix Inversion ------------------------------------------- */
00337 /*
00338    The two matrix inversion functions work on rows*rows matrices of
00339    ints.  If a bitmatrix, then each int will just be zero or one.
00340    Otherwise, they will be elements of gf(2^w).  Obviously, you can
00341    do bit matrices with crs_invert_matrix() and set w = 1, but
00342    crs_invert_bitmatrix will be more efficient.
00343 
00344    The two invertible functions return whether a matrix is invertible.
00345    They are more efficient than the inverstion functions.
00346 
00347    Mat will be destroyed when the matrix inversion or invertible
00348    testing is done.  Sorry.
00349 
00350    Inv must be allocated by the caller.
00351 
00352    The two invert_matrix functions return 0 on success, and -1 if the
00353    matrix is uninvertible.
00354 
00355    The two invertible function simply return whether the matrix is
00356    invertible.  (0 or 1). Mat will be destroyed.
00357  */
00358
